HOW DOES LSB STEGANOGRAPHY WORK?

A steganographer conceals the secret information in the least significant bits of a media file in this type of steganography. An image file, for instance, consists of three bytes of data for each pixel corresponding to red, green, and blue (some image formats assign a fourth byte to transparency). The last bit of each of those bytes is changed in LSB steganography so that one bit of data is hidden. For this method to hide one megabyte of data, an eight-megabyte image file is required. Changing the last bit of a pixel value does not result in a visually noticeable change to the image, so viewing the original and the steganographically modified images won't reveal any differences.

DIFFERENCE BETWEEN STEGANOGRAPHY AND CRYPTOGRAPHY

Steganography hides information, while cryptography encrypts it so that no one except the intended recipient can read it. The only way to decrypt a stream of data is with its decryption key, which is only accessible to those who possess it.
